## 项目简介

### 目标

本着学习的目的，试图构建一个更加简洁、优雅的、Pythonic 的知乎 API。


## 前置条件

* Python3.x
* Requests2
* BeautifulSoup4

## 安装

```python
git@github.com:lzjun567/zhihu-api.git
cd  zhihu-api
pip install -r requirement.txt
```
